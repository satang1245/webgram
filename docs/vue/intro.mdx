## Vue란?
Vue (view의 의미로  /vjuː/ 라고 발음합니다)는 사용자 인터페이스를 구축하기 위한 `자바스크립트 프레임워크`이다.

### 특징

#### 1. 선언적 렌더링(Declarative Rendering)
- Vue는 표준 HTML을 기반으로 확장한 템플릿 문법으로 Javascript 상태(State)를 기반으로 화면에 출력될 HTML을 선언적(declaratively)으로 작성할 수 있다.  
즉, HTML의 확장된 템플림을 말하며, 해당 문법은 HTML로 보여지지만, Javascript로 제작된 템플릿 언어이다.

#### 2. 반응성(Reactivity)
- Vue는 JavaScript 상태 변경을 자동으로 추적하고 변경이 발생하면 효율적으로 DOM을 업데이트합니다.

```javascript
import { createApp, ref } from 'vue'

createApp({
  setup() {
    return {
      count: ref(0)
    }
  }
}).mount('#app')
```

```html
<div id="app">
  <button @click="count++">
    Count is: {{ count }}
  </button>
</div>
```


---

Vue는 프론트엔드 개발에 필요한 대부분의 공통 기능을 다루는 프레임워크 입니다.  
하지만 웹은 매우 다양합니다. 웹에서 구축하는 것들은 형태와 규모가 크게 다를 수 있습니다.  
이를 염두에 두고 Vue는 유연하고 점진적으로 채택 가능하도록 설계되었습니다.  
사용 사례에 따라 Vue를 다양한 방식으로 사용할 수 있습니다.

- 모든 페이지에 웹 구성 요소로 포함
- 단일 페이지 애플리케이션(SPA: Single-Page Application)
- 풀스택/서버 사이드 렌더링(SSR: Server-Side-Rendering)
- Jamstack / 정적 사이트 생성(SSG: Static-Site-Generation)
- 데스크탑, 모바일, WebGL, 심지어 터미널까지 타겟팅  

[용어설명](https://vuejs.org/guide/extras/ways-of-using-vue)

### Single-File Components (SFC, 단일 파일 구성요소)

빌드 도구를 사용하는 대부분의 Vue 프로젝트에서는 `*.vue` 라는 **HTML과 유사한 파일 형식**을 가진 SFC 파일을 작성하여 사용합니다.  
Vue SFC는 구성요소의 로직(Javascript), 템플릿(HTML), 스타일(CSS)을 단일 파일에 캡슐화 하여 사용됩니다.  
```html
<script>
export default {
  data() {
    return {
      count: 0
    }
  }
}
</script>

<template>
  <button @click="count++">숫자 세기: {{ count }}</button>
</template>

<style scoped>
button {
  font-weight: bold;
}
</style>
```
#### 왜? why SFC?!
관심사항의 분리.  
**장점**
- 친숙한 HTML, CSS 및 JavaScript 문법을 사용하여 모듈화된 컴포넌트 작성
- 본질적으로 사용 목적에 따라 구성됨
- 런타임 컴파일 비용이 없는 사전 컴파일된 템플릿
- 컴포넌트 범위 CSS
- 컴포지션 API로 작업할 때 더욱 인체공학적인 문법
- 템플릿과 스크립트를 교차 분석하여 컴파일 시간을 더욱 더최적화
- 템플릿 표현식을 지원하는 IDE의 자동 완성 및 유형 검사
- 즉시 사용 가능한 핫 모듈 교체(Hot-Module Replacement: HMR) 지원

Vue SFC는 프레임워크 파일 형식이며, 해당 파일을 Javascript 및 css로 미리 컴파일 하여  
Javascript(ES) 모듈처럼 사용한다.
```js
import MyComponent from './MyComponent.vue'

export default {
  components: {
    MyComponent
  }
}
```
:::info
*컴파일을 통해 `<style>`요소는 별도의 css파일로 추출 및 병합 할 수 있다.  
:::

### 2. CDN 서버를 통해 vue.js를 로드하는 방법

```html
<!-- vue 전역객체 로드 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>
<script>
  const { createApp, ref } = Vue

  createApp({
    setup() {
      const message = ref('Hello vue!')
      return {
        message
      }
    }
  }).mount('#app')
</script>


<!-- ES module 사용시 -->
<div id="app">{{ message }}</div>
<script type="module">
  import { createApp, ref } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

  createApp({
    setup() {
      const message = ref('Hello Vue!')
      return {
        message
      }
    }
  }).mount('#app')
</script>
```
