# 타입정의

typescript에서는 변수나 인수명 뒤에 : 타입과 같은 <u>타입 애너테이션</u>이라 불리는 타입 정보를 붙여준다.

## 선언방법  
- `변수`: `타입` = `값`

```ts


// String
let userName: string = 'satang1245'

// Number
let userAge: number = 36

// boolean
let isSelected: boolean = false

// 뭐든지 - 어떤 타입이 와야 하는지 명확하지 않을 때 사용. 
// 이걸 쓰면 타입스크립트 사용하는 의미가 없어지기 때문에, 잘 생각해보고 사용하는게 좋다.
let hello: any = 'world' 

// 다중표현.
let element: string | HTMLElement = 'userId'
```
### 배열의 경우
 - 배열의 경우 2가지 방식으로 타입 선언이 가능하다.

```ts
// 배열 선언방법 1.
let arr: string[] = ['H','e','l','l','o']
// 배열 선언방법 2. 
let arr: Array<number> = = [24,12,16,42,1,5,31]

// Union (유니온)타입
let mixArr: (string|number)[] = ['satang1245', 36]
// Tuple (튜플)타입 - 정해진 갯수만큼 타입을 미리 지정
let mixArr: [stirng,number] = ['satang1245', 36]
``` 

### 객체의 경우
```ts
// { key: 타입; key: 타입 }
const user = { name: string; age: number } = {
	name: 'satang1245',
	age: 36
}

// 옵셔널 (Optional) - 객체 타입에 일부 키값이 선택적일 경우
const user = { name: string; age: number; phone?: string } = {
	name: 'satang1245',
	age: 36
}
```

### 함수의 경우

```ts
// function 함수명( 인수: 타입, 인수: 타입 ): 반환값 {
// ,,,
// }
function sayHello(firstName: string, lastName: string): string {
	return `Hello ${lastName} ${firstName}`
}
// 옵셔널 인수도 가능.
const sayHello = function(firstName: string, lastName?: string): string {
	return `Hello ${lastName} ${firstName}`
}

// case : arrow function
// 인수에 기본값 할당 가능. (age = 21)
const sayHello = (name: string, age: number = 21):string => {
	return `안녕하세요 ${name}님. 님 나이는 ${age}살 입니다` 
}

// 반환값이 없거나 return문이 없을 때 : void
function greet(name: string): void {
	console.log("hello," + name.toUpperCase())
}
```

## 타입종류

typescript는 ECMAScript의 확장 언어로써, es5 및 es6에서 사용되던 타입 이외에도  
typescript 고유의 타입이 추가 존재한다.

| Type | JS | TS | Description | 
| :--: | :--: | :--: | :-- | 
| boolean | ◯ | ◯ | true와 false | 
| null | ◯ | ◯ | 값이 없다는 것을 명시 | 
| undefined | ◯ | ◯ | 값을 할당하지 않은 변수의 초기값 | 
| number | ◯ | ◯ | 숫자(정수와 실수, Infinity, NaN) | 
| string | ◯ | ◯ | 문자열 | 
| symbol | ◯ | ◯ | 고유하고 수정 불가능한 데이터 타입이며 주로 객체 프로퍼티들의 식별자로 사용(ES6에서 추가) | 
| object | ◯ | ◯ | 객체형(참조형) | 
| array |  | ◯ | 배열 | 
| tuple |  | ◯ | 고정된 요소수 만큼의 타입을 미리 선언후 배열을 표현 | 
| enum |  | ◯ | 열거형. 숫자값 집합에 이름을 지정한 것 | 
| any |  | ◯ | 타입 추론(type inference)할 수 없거나 타입 체크가 필요없는 변수에 사용, var 키워드로 선언한 변수와 같이 어떤 타입의 값이라도 할당 가능 | 
| void |  | ◯ | 일반적으로 함수에서 반환값이 없을 경우 사용 | 
| never |  | ◯ | 결코 발생하지 않는 값|

let lens: number = (<string>myNickname).length
// or `as`문법을 사용하는 경우
let lens: number = (myNickname as string).length
