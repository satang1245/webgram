## Framework(프레임워크)
`완제품` 개발자가 필요한 코드의 기틀과, 문법, 라이브러리 등을 기본 제공하는 것  
대표적으로 Vue가 있다.

## Library(라이브러리)
`부품` 클래스 또는 함수들의 나열로 사용자가 필요한 부분을 사용이 가능하며,  
별도의 라이브러리를 추가추가하여 완성품을 만드는 것.  
프레임워크에 비해 자유도가 높기 때문에 처음 진입장벽은 높지만,  
다양한 방향으로 사용이 가능하다는 장점이 있다.  
대표적인 라이브러리는 React, jQuery 등이 있다.

---

## Vue Framework vs React library
앞서 말한바와 같이 뷰 프레임워크는 지정된 문법, 즉 .vue파일내에  
&lt;script&gt;안에는 js, templete안에 html, style 안에서는 css를 작성해야 하지만,  
리액트 라이브러리의 경우 JSX(Javascript XML)형태의 코드를 작성하기 때문에,  
javascript만으로 UI 로직과 DOM을 구현가능하다.
