# 변수 & 상수 & 데이터 타입

## 변수 (variable)
변수는 하나의 값을 저장하기 위해 확보한 메모리 공간 자체 또는 그 메모리 공간을 식별하기 위해 붙인 이름.

### 변수는 왜 사용하는가?
```javascript
var userId = 1; // 할당 (변수명 = 변수값)
var userName = 'Lee';
```
변수는 연산된 결과값을 재사용 할 수 있도록 저장된 메모리 공간에 상징적인 이름을 붙인 것이다.   
*변수명은 *식별자* 라고도 불린다.

### 변수는 어디에 저장되는가?
변수는 자바스크립트 엔진내 관리영역인실행 `컨텍스트`에 저장되며,  
실행 컨텍스트 내에 **키(key)** / **값(value)** 형식인 객체로 등록되어 관리된다.

---

### 값
프로그램에 의해 조작될 수 있는 대상.  
```javascript
var str = 'Hello world!'; 
```
위 소스에서 `str`은 **변수**, `hello world!`는 문자열 리터럴의 **값**을 의미한다. 

| 용어 | 의미 |
| :---  | :--- |
| 데이터 타입(Data type) | 프로그래밍 언어에서 사용할 수 있는 값의 종류 <br /> (Number, String, Boolean, null, undefined, Symbol) |
| 변수(variable) | 값이 저장된 메모리 공간의 주소를 가리키는 식별자(indentifier) |
| 리터럴(literal) | 소스코드 안에서 직접 만들어 낸 값 자체를 말하며, 값을 구성하는 최소 단위 |

### 문
문이란, 컴퓨터에 의해 프로그램인 스크립트가 단계별로 실행되는 명령들의 집합을 의미하며,   
리터럴, 연산자, 표현식, 키워드 등으로 구성되고 세미콜론(;)으로 끝나야 한다.  

<small>*문의 끝에 붙이는 세미콜론은 자바스크립트 엔진이 스크립트를 해석할 때  
문의 끝이라고 예측되는 지점에 세미콜론을 자동으로 붙여주는 세미콜론 자동 삽입 기능이 포함되어 있다.  
하지만 세미콜론 자동 삽입 기능의 예측과 개발자의 예측과 다른 경우가 간혹 있어서 사용에 주의해야 한다.</small>

---

### 변수 선언 및 할당
```javascript
var str;  // 변수 선언
str = 'Hello World';  // 변수 할당
str = 'Nice to meet you!'; // 변수 재할당

// 변수 선언 및 할당
var person = 'Son',
    address = 'Seoul',
    age = 29;

var person, address, age; //변수 선언
person = 'Son';         // 변수 할당
address = 'Seoul';      // 변수 할당
age = 29;               // 변수 할당
```

## 상수 (constant)
변수는 값의 재할당이 가능한 것을 말하며,  
상수는 **값의 재할당이 불가능한 것**을 말한다.  
*단, `const` 키워드로 선언된 변수가 객체가 아닐 경우, 값을 변경 할 수 없다.  
const 키워드는 재할당을 금지할 뿐 "**불변**"을 의미하지 않는다.
```javascript
// [X] : const 키워드는 재할당이 불가. 
const str = 'hello';
str = 'Hi ~ 👋🏻'; // Uncaught TypeError: Assignment to constant variable.

// [O] : 객체일 경우 값 변경 가능.
const user = {
    name : '바나나',
}
// 변수에 직접적으로 재할당은 불가능하지만, 객체내 키값을 재할당 및 신규 추가 및 제거 가능.
user.name = '파인애플';
user.age = 21;

// [X] - 재할당은 불가.
user = {name: '사과', age: 32} // Uncaught TypeError: Assignment to constant variable.
```

## 데이터 타입 (data type)

### 원시타입 (primitive type)
원시 타입의 값은 **변경 불가능한 값**이며, 값에 의한 전달이다.

#### number
C언어 또는 Java등 일반적인 프로그램언어에서는 정수, 실수를 구분하여  
Int, long, float, double 등과 같은 여러 형태의 숫자 타입이 존재하지만,  
자바스크립트는 독특하게 하나의 숫자타입(Number)만 존재한다.  
<small>*(Chrome 67버전 이후 [BigInt](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/BigInt) 타입이 추가되었다.)</small>

```javascript
// 모든 숫자는 표기법만 다를 뿐 모두 정수가 아닌 실수로 표기한다.
var integer = 10;        // 정수
var double = 10.12;      // 실수
var negative = -20;      // 음의 정수
var binary = 0b01000001; // 2진수
var octal = 0o101;       // 8진수
var hex = 0x41;          // 16진수

console.log(binary); // 65
console.log(octal);  // 65
console.log(hex);    // 65

// 표기법만 다를뿐 같은 값이다.
console.log(binary === octal); // true
console.log(octal === hex);    // true

// 정수로 표시되지만 실제 처리는 실수로 처리한다.
console.log(1 === 1.0); // true

var pInf = 10 / 0;  // 양의 무한대
console.log(pInf);  // Infinity

var nInf = 10 / -0; // 음의 무한대
console.log(nInf);  // -Infinity

var nan = 1 * 'string'; // 산술 연산 불가
console.log(nan);       // NaN
```

#### string
문자열(String) 타입은 텍스트 데이터를 나타내는데 사용하며,  
작은 따옴표(') 또는 큰 따옴표(") 안에 텍스트를 넣어 생성한다.
```javascript
var str = 'Hello';

// 문자열은 유사배열이다.
for (var i=0; i < str.length; i++) {
    console.log(str[i]);
}

// 배열이 아닌 유사배열이기 때문에 배열 방식을 통하여 문자열을 변경할 수 없다.
str[0] = 'S';
console.log(str); // Hello

// 문자열 연산 및 재할당이 가능
str += ' world'; // Hello world
```

#### boolean
논리 타입으로 `true` 또는 `false`만 표현 가능하다.  

:::tip Truthy 값  
`false`, `undefined`, `null`, `0`, `NaN`, `""(빈 문자열)` 을 **제외한 모든 경우**.  
:::

#### undefined
undefined 타입의 경우 `undefined`의 유일한 값을 가진다.  
변수 선언 이후 별도의 값을 할당하지 않거나, 존재하지 않는 객체 프로퍼티에 접근할 경우 해당값을 반환한다.
```javascript
var foo;
console.log( foo ); // undefined

 // 인수를 아무것도 넣지 않고 함수를 실행할 경우,
 // 전달받는 매개변수는 undefined가 된다.
function hello( str ) {
    console.log(str); // undefined
}
hello();
```

#### null
null 타입의 경우 `null`의 유일한 값을 가진다.  
변수의 값이 없다는 것을 명시적으로 표현할 때 사용하며,  
함수나 DOM 검색등에서 유효한 값을 반환할 수 없을 경우 null을 반환한다.  
<small>*자바스크립트의 경우 대소문자를 구분하기 때문에, Null이나 NULL등과 다름으로 사용에 유의해야 한다.</small>
```javascript
var element = document.querySelector('.button');

// HTML 문서에 button 클래스를 갖는 요소가 없다면 null을 반환한다.
console.log(element); // null
```

#### Symbol
ES6에서 새롭게 추가된 타입으로, 고유한 프로퍼티 키(property key)를 만들기 위해 사용된다.  
```javascript
var key = Symbol('key'); // Symbol(key)
var obj = { 'key' : 'value' }; // obj = { key: 'value' }

obj['key'] == 'value2'; // obj = { key: 'value2' }
obj[key] == 'value3'; // obj = { key: 'value2', Symbol(key): 'value3' }
```

### 객체 타입
자바스크립트는 객체 지향 프로그래밍으로 설계된 언어로,  
원시 타입을 제외한 모든 값(`배열`, `함수`, `정규표현식`)을 객체라 한다.  
<small>*객체는 키(key)와 값(value)로 이루어진 프로퍼티 들의 모음을 말한다.</small>
