웹 브라우저의 구성요소들은 각각 객체화 되어 있으며, 이러한 객체들은 서로 계층적인 관계로  
구조화 되어 있다. 해당 구조는 위 그림과 깉이 브라우저 최상의 객체인 window객체를 기준으로,  
DOM객체, BOM객체, Core(ECMAScript) 로 구성되어 있다.  

## 1. DOM (Document Object Model)
문서 모델 객체(Dcoument Object Model)는 웹 페이지내의 모든 콘텐츠를 객체로 나타내는데,  
문서는 노드(Node)라고 불리는 계층적 단위로 구성되어 있다.  
또한 노드는 문서(document node), 요소(element node), 속성(attribute noe), 텍스트(text node), 주석(comment node)로 분류되어 있다.

## 2. BOM (Browser Object Model)
문서(웹페에지)를 제외한 브라우저를 제어하기 위한 객체들의 모임.  
브라우저의 정보를 통하여, 디바이스 또는 브라우저 종류 및 버전에 따른 예외 처리가 가능하며,  
창 사이즈의 변환이나 페이지 이동등 문서를 제외한 여러가지를 제어할 수 있다.
